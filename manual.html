1. 概要

Kawa Code エコシステムは、複数の相互接続されたコンポーネントで構成されています。

コンポーネント

Muninn

Kawa Code デスクトップアプリケーション - Git 操作を管理し、差分を生成し、クラウドと同期する中心的なハブ

Muninn Extensions

kawa.i18n コード翻訳拡張機能など、Kawa Code メインアプリケーション用の拡張機能

Huginn Extensions

Kawa Code に接続し、コラボレーション機能を表示するエディタプラグイン (VSCode、Emacs、Vim)

Kawa API

チームメンバーを調整するクラウドサービス

MCP Server

インテント追跡機能を備えた AI 支援開発のための Claude Code との統合

アーキテクチャ

Editor (VSCode/Emacs/Vim)
    ↓
Unix sockets / Named pipes
    ↓
Muninn (Desktop App)
    ↓
WebSockets
    ↓
Kawa API (Cloud)

データストレージと同期

  • ローカルストレージ：すべてのデータはローカルマシンに保存されます
  • クラウド同期：オプションでチームメンバーとの同期が可能
  • リアルタイム更新：WebSocketsを使用した即座の変更通知

2. システム要件

macOS

  • macOS 11.0 (Big Sur) 以降
  • Apple Silicon (M1/M2/M3) または Intel プロセッサ
  • 4GB RAM 以上推奨

Windows

  • Windows 10 (64-bit) 以降
  • 4GB RAM 以上推奨
  • .NET Framework 4.7.2 以降

Linux

  • Ubuntu 20.04 以降 / Debian 11 以降
  • 4GB RAM 以上推奨
  • GTK+ 3.0 以降

4. Muninn 拡張機能

拡張機能のインストール

Muninnアプリケーション内から、拡張機能をインストールできます。

  1. Muninnアプリを起動
  2. 設定 > 拡張機能 に移動
  3. 利用可能な拡張機能から選択してインストール

推奨拡張機能

  • kawa.i18n - コード翻訳機能（日英変換）
  • kawa.ai-audit - AI生成コードの監査機能
  • kawa.intent-tracker - 開発意図の追跡

独自の拡張機能の作成

Kawa Code SDKを使用して、独自の拡張機能を開発できます。

# SDK のインストール
npm install @kawacode/sdk

# プロジェクトのセットアップ
kawa-cli create-extension my-extension

6. CLAUDE.md のセットアップ

プロジェクトルートにCLAUDE.mdファイルを作成することで、AIアシスタントに開発意図を伝えることができます。

CLAUDE.md テンプレート

# プロジェクト概要
このプロジェクトは...

# 開発ガイドライン
- コーディング規約
- アーキテクチャの原則
- テスト方針

# 現在の課題
- 実装中の機能
- 解決すべき問題

AIとの連携

CLAUDE.mdを設定することで、AIアシスタントがプロジェクトのコンテキストを理解し、より適切な提案を行えます。

7. トラブルシューティング

Muninnが起動しない

解決策：

  • アプリケーションを再インストール
  • システム要件を確認
  • ログファイルを確認（~/.kawa/logs）

エディタ拡張機能が接続できない

解決策：

  • Muninnアプリが起動しているか確認
  • 拡張機能を再インストール
  • ファイアウォール設定を確認

同期エラー

解決策：

  • インターネット接続を確認
  • アカウント認証を確認
  • キャッシュをクリア
